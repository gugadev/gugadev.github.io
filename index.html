Proyectos

COPECADE (Frontend)

Este proyecto constaba del diseño, análisis e implementación de la parte Frontend, ya que la empresa ya contaba tanto como un esqueleto básico hecho con CodeIgniter (PHP) por trabajadores anteriores como con una API local conectada a MySQL para interactuar con la base de datos. Para el prototipo usé Photoshop (básico) y Balsamiq. Para la implementación, usé Sass como preprocesadores CSS, y Vue.js 2 como framework cliente. Para el panel de administración, usé Vue Router. Toda la parte cliente se desarrolló usando ES6 transpilando con Gulp.

Optima Network HQ

Las aplicaciones que realicé aquí, todas web, fueron realizadas con Node.js debido al alto tráfico que ganábamos de los ad servers. La elección de Node.js y NET Core (Web API) fue debido a la alta capacidad por manejar grandes cantidades de peticiones debido a su alta concurrencia y la robustez de la arquitectura NET. Las aplicaciones usaron distintas tecnologías, entre ellas: React.js, Vue.js, React Router, Vue Router, Redux y Vuex como frameworks UI. Para la implementación de la UI a nivel de estilos, se usó PostCSS. Todas las aplicaciones, en la parte cliente se usó ES6+ y fueron transpiladas usando Webpack. La arquitectura de las aplicaciones fueron, algunas orientadas a servicios. La mayoría de APIs fueron RESTful e implementadas en NET Core Web API y Koa.js, sin embargo, el último proyecto se usó GraphQL debido a su facilidad para consultas y su validación integrada.
